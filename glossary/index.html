HTML Attributes Glossary

Working with text

  • h1/2/3/etc. - Headline tags used for titles
  • p - contains a paragraph
  • b - bold text
  • i/em - italicised text
  • sup - superscript characters, such as e = mc2
  • sub - subscript characters, such as logn
  • Entity codes - used for special characters, such as №
  • a href - creating hyperlinks

Other structures

  • ul - creating an unordered (bulleted) list
  • ol - creating an ordered (numbered) list
    1. One
    2. Two
    3. Three
  • li - individual list elements
  • img - adding images (no closing tag used)
